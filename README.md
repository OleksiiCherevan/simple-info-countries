### To run this application you should use the following commands: 

* ?if you don`t have yarn
```bash
   npm install -g yarn 
```
* clone this project 
```bash
   git clone <"this_repo_url">
```

* open the directory with the project
```bash
   cd <"this_repo_url">
```

* install dependencies
```bash
   yarn
```

* run application
```bash
    yarn start
```
